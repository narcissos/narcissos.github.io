Future Endeavours

Currently, I'm in the process of creating my own visual novel.

A visual novel is an interactive game combining text, visuals, music, and of course, a storyline. To make such a game, I had to teach myself a program called Ren'Py + its code language, as well as Python. Other software like Twine and Unity taught me how to navigate the complexities of web language, but it was no easy task to switch gears to Ren'Py.

More than anything, creating the vast amount of visuals to go with the story, picking appropriate music, and making sure it all felt consistent with each other was the hard part. My demo is nearly complete so I hope to keep growing as an amateur game dev during Summer 2021.
